HTML Minifier Pro supports HTML, CSS, and JavaScript files. You can minify each type individually or work with combined HTML files that contain inline CSS and JavaScript.

Absolutely. All processing happens securely on our servers. Your code is never stored permanently, logged, or shared with third parties. We process and return results immediately.
